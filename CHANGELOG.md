# Changelog

## [0.1.0] - 2026-02-15

### Added
- **New Embedding Syntax**: `<!-- embed:file="..." ... -->` (HTML comments) replaces the default output-visible syntax.
- **Region Support**: Added `#region` support for embedding specific code blocks.
- **Demo**: Added `demo.md` and `src/demo.ts` to showcase features.
- **License**: Added MIT License.

### Changed
- **Refactoring**: Split code into `extension.ts`, `embedder.ts`, and `utils.ts` for better maintainability.
- **Documentation**: Updated `README.md` with new features and usage instructions.
